# Как найти настройку Security Level в Cloudflare

## Где находится Security Level

Security Level находится не в модальном окне "Settings", а на главной странице Security → Settings.

## Пошаговая инструкция

### Вариант 1: Через главную страницу Security Settings

1. **Закройте модальное окно** (нажмите X или вне окна)
2. На главной странице **Security → Settings** вы увидите секцию **"Security Level"**
3. Там будет выпадающий список или переключатель с опциями:
   - **Off** (выключено)
   - **Essentially Off** (практически выключено)
   - **Low** (низкий) ← **выберите это**
   - **Medium** (средний)
   - **High** (высокий)
   - **I'm Under Attack!** (режим атаки)

4. Выберите **"Low"** или **"Essentially Off"** для тестирования
5. Сохраните изменения

### Вариант 2: Если не видите Security Level на главной странице

1. В Cloudflare Dashboard → **Security** → **Settings**
2. Прокрутите страницу вниз
3. Найдите секцию **"Security Level"** (она может быть ниже других настроек)
4. Или найдите кнопку **"Change"** рядом с текущим уровнем безопасности

## Альтернативный способ: через WAF

Если Security Level не виден, можно проверить через WAF:

1. Cloudflare Dashboard → **Security** → **WAF**
2. Проверьте настройки **Managed Rules**
3. Временно отключите правила, которые могут блокировать запросы

## Что делать дальше

После изменения Security Level на "Low":

1. **Очистите кеш Cloudflare:**
   - Caching → Configuration → Purge Everything

2. **Проверьте Speed Optimization:**
   - Speed → Optimization
   - Отключите Auto Minify и Rocket Loader

3. **Подождите 5 минут** и проверьте сайт в режиме инкогнито

## Если Security Level не найден

Если вы не можете найти настройку Security Level, это может означать:

1. **У вас Free план** - некоторые настройки могут быть ограничены
2. **Настройка находится в другом месте** - попробуйте поиск в Cloudflare Dashboard

В этом случае сосредоточьтесь на:
- Отключении Auto Minify и Rocket Loader (Speed → Optimization)
- Очистке кеша
- Проверке WAF правил

## Быстрая проверка

Попробуйте найти в интерфейсе Cloudflare:
- Поиск по "Security Level" (если есть поиск)
- Или перейдите: Security → Settings → прокрутите вниз

Если не найдете - не критично, сосредоточьтесь на других настройках из чеклиста.
