# Инструкции по добавлению Lottie анимации с персонажем

## Требования к анимации

- ✅ **Персонаж в Telegram стиле** (утёнок или похожий)
- ✅ **Визуально объясняет идею сервиса** (продуктивность, задачи, организация)
- ❌ Запрещено: круги, абстрактные фигуры, лоадеры, геометрические заглушки

## Как найти подходящую анимацию

### 1. LottieFiles.com

1. Открой https://lottiefiles.com
2. В поиске введи: `duck`, `telegram duck`, `productivity`, `task management`
3. Фильтры:
   - **Free** (бесплатные)
   - **JSON** формат
   - **Character** или **Illustration** категория

### 2. Рекомендуемые поисковые запросы

- `duck character`
- `telegram style duck`
- `cute duck animation`
- `productivity duck`
- `task management character`

### 3. Примеры подходящих анимаций

Ищи анимации, где персонаж:
- Машет рукой / приветствует
- Работает с задачами / списками
- Показывает продуктивность
- Дружелюбный и позитивный

## Как добавить анимацию

### Вариант 1: Локальный файл (рекомендуется)

1. Найди подходящую анимацию на LottieFiles.com
2. Скачай JSON файл
3. **Замени файл:** `src/assets/welcome-animation.json` (текущий файл - заглушка)
4. Готово! Анимация загрузится автоматически при следующем запуске

**Важно:** Файл уже существует как заглушка. Просто замени его содержимое на скачанный JSON.

### Вариант 2: Загрузка по URL

1. Найди анимацию на LottieFiles
2. Скопируй прямую ссылку на JSON файл
3. Открой `src/components/Welcome.tsx`
4. Раскомментируй код загрузки по URL (строки 20-25)
5. Вставь ссылку вместо `'https://lottie.host/embed/...'`

Пример:
```typescript
fetch('https://lottie.host/embed/abc123.json')
  .then(res => res.json())
  .then(data => setAnimationData(data))
```

## Проверка

После добавления анимации:
1. Запусти `npm run dev`
2. Открой приложение в Telegram
3. Проверь Welcome экран
4. Анимация должна автоматически проигрываться

## Размеры анимации

Анимация отображается в размере **280x280px** для лучшей видимости персонажа.

Если анимация слишком большая или маленькая, измени размер в `Welcome.tsx`:
```typescript
width: '280px',  // измени на нужный размер
height: '280px',
```

## Troubleshooting

**Анимация не загружается:**
- Проверь, что файл находится в `src/assets/`
- Проверь формат файла (должен быть валидный JSON)
- Открой консоль браузера для ошибок

**Анимация не отображается:**
- Убедись, что JSON файл содержит правильную структуру Lottie
- Проверь, что `lottie-react` установлен: `npm install lottie-react`

**Персонаж не подходит:**
- Попробуй другие поисковые запросы на LottieFiles
- Или создай свою анимацию в After Effects + Bodymovin
