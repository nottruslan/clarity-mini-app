# Инструкция по размещению логотипа

## Проблема
Логотип не отображается на splash screen - показывается только буква "C" (fallback).

## Решение

### Шаг 1: Подготовьте файл логотипа
1. Убедитесь, что у вас есть файл логотипа в формате PNG
2. Файл должен быть с прозрачным фоном (если нужно) или на белом фоне

### Шаг 2: Разместите файл в папке `public`
1. Откройте папку проекта: `/Users/ru/Desktop/clarity — копия/`
2. Найдите папку `public/`
3. Поместите файл логотипа в эту папку с именем **`logo.png`** (обязательно именно такое имя!)

**Важно:**
- Имя файла должно быть точно `logo.png` (без двоеточий, пробелов и других специальных символов в начале)
- Файл должен быть в формате PNG
- Путь: `public/logo.png`

### Шаг 3: Проверьте структуру
После размещения файла структура должна быть такой:
```
clarity — копия/
├── public/
│   ├── logo.png        ← ваш логотип (новый файл)
│   ├── anim.json
│   └── :log.png        ← старый файл (можно удалить)
└── ...
```

### Шаг 4: Перезапустите приложение
1. Остановите dev-сервер (если запущен)
2. Запустите заново: `npm run dev`
3. Или пересоберите проект: `npm run build`

### Проверка
После размещения файла логотип должен отображаться на splash screen при загрузке приложения.

## Альтернативные пути (если не работает)
Если файл по пути `/logo.png` не загружается, код автоматически попробует альтернативные пути. Но рекомендуется использовать именно `/logo.png`.

## Требования к файлу
- Формат: PNG (рекомендуется)
- Размер: рекомендуется квадратное изображение (например, 512x512 или 1024x1024 пикселей)
- Фон: прозрачный или белый (в зависимости от вашего дизайна)
- Размер файла: желательно до 500KB для быстрой загрузки
